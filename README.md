# 自己紹介サイト

## 概要
このプロジェクトは、HTML/CSS/JavaScriptを用いて25JZ0113についての情報を紹介しています。

## ファイル構成
```
自己紹介サイト/
├── index.html          # メインページ
├── favicon.ico         # ファビコン
├── css/
│   ├── reset.css       # リセットスタイル
│   ├── style.css       # メインスタイル
│   └── responsive.css  # レスポンシブスタイル
├── js/
│   └── script.js       # JavaScriptコード
├── img/                # 画像ファイル
├── contact/
│   └── index.html      # お問い合わせページ
└── README.txt          # このファイル
```

## 主な機能
- **レスポンシブデザイン**: モバイル/タブレット/デスクトップに対応
- **マルチページ対応**: メインページとお問い合わせページを搭載
- **動的な要素**: JavaScriptを使用したインタラクティブな機能
- **グリッドレイアウト**: CSSグリッドを活用した整理されたレイアウト

## ページ構成
- **トップページ (index.html)**: プロフィール、スキル、趣味などの情報を掲載
- **お問い合わせページ (contact/index.html)**: 連絡フォーム

## 技術スタック
- HTML5
- CSS3 (FlexBox, Grid, Media Queries)
- JavaScript (ES6+)
- Google Fonts

## ブラウザ対応
主流のモダンブラウザ（Chrome, Firefox, Safari, Edge）に対応しています。

## 使用方法
1. ファイルをローカルに保存
2. index.html をブラウザで開く
3. サイトを閲覧・操作

## 参考資料
